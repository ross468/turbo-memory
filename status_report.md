# 3DS 윈도우 7 시뮬레이터 프로젝트 진행 현황

## 1. 현재 진행 상황
- **UI 에셋 준비**: 윈도우 7 배경화면 및 시작 버튼 아이콘을 수집하고 3DS 화면 규격(400x240, 320x240)에 맞게 리사이징 완료했습니다.
- **소스코드 작성**: C++와 citro2d 라이브러리를 기반으로 한 기본 시뮬레이터 코드를 작성했습니다.
  - 상단 화면: 윈도우 7 배경 및 작업 표시줄
  - 하단 화면: 배경, 작업 표시줄, 시작 버튼 및 터치로 조작 가능한 마우스 커서 구현
- **빌드 환경 구성 중**: 현재 샌드박스 환경에서 3DS 컴파일러(`devkitPro`) 설치를 시도 중입니다. 외부 저장소 접근 제한으로 인해 빌드에 다소 시간이 소요되고 있습니다.

## 2. 향후 계획
- 빌드 환경 문제를 해결하여 실제 실행 가능한 `.3dsx` 파일을 생성할 예정입니다.
- 추가적인 윈도우 7 기능(시작 메뉴 팝업, 간단한 메모장 등)을 구현할 계획입니다.

## 3. 기술적 세부 사항
- **언어**: C++
- **라이브러리**: libctru, citro2d, citro3d
- **대상**: Nintendo 3DS (Homebrew Launcher용)
