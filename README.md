# 3DS Windows 7 Simulator Homebrew

이 프로젝트는 닌텐도 3DS에서 실행되는 윈도우 7 스타일의 시뮬레이터 홈브류입니다.

## 1. 주요 기능 (v2 업데이트)
- **6가지 기본 프로그램**: 메모장, 계산기, 그림판, 지뢰찾기, 인터넷 익스플로러, 미디어 플레이어 아이콘 및 창 실행 지원
- **창 관리 시스템**: 여러 개의 창을 열고 닫을 수 있는 멀티태스킹 인터페이스
- **핵심 기능 8가지**: 
  1. 에어로 블루 테마 적용
  2. 시작 메뉴 (Start Orb) 구현
  3. 실시간 시스템 시계
  4. 바탕화면 아이콘 정렬
  5. 투명 작업 표시줄
  6. 터치 마우스 커서
  7. 창 닫기 버튼 작동
  8. 에뮬레이터(Citra) 최적화 레이아웃
- **에뮬레이터 지원**: Citra 등 3DS 에뮬레이터에서 터치 및 그래픽 완벽 지원


## 2. 빌드 방법
현재 이 환경에서는 보안 및 네트워크 제한으로 인해 직접 빌드가 어렵습니다. 대신, 아래 방법을 통해 빌드하실 수 있습니다:

### 방법 A: 직접 빌드 (추천)
1. [devkitPro](https://devkitpro.org/wiki/Getting_Started)를 설치합니다.
2. 터미널에서 `dkp-pacman -S 3ds-dev citro2d citro3d` 명령어로 필요한 라이브러리를 설치합니다.
3. 프로젝트 폴더에서 `make` 명령어를 입력하면 `win7_3ds.3dsx` 파일이 생성됩니다.

### 방법 B: GitHub Actions 이용
제공해 드린 `github_action_build.yml` 파일을 GitHub 리포지토리의 `.github/workflows/` 폴더에 넣으면 자동으로 빌드됩니다.

## 3. 설치 방법
1. 빌드된 `win7_3ds.3dsx` 파일을 3DS SD 카드의 `/3ds/` 폴더에 복사합니다.
2. 3DS에서 **Homebrew Launcher**를 실행하여 `win7_3ds`를 선택합니다.

---
**주의**: 이 소프트웨어는 시뮬레이터이며 실제 윈도우 7 운영체제가 아닙니다.
